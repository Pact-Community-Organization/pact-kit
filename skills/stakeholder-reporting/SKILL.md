# Stakeholder Reporting

## Sprint Review Template
```markdown
## Sprint {N} Review — {date range}

### Completed
- {story}: {brief result} ✅
- {story}: {brief result} ✅

### In Progress
- {story}: {current status, blockers}

### Metrics
- Stories completed: {N} of {M}
- Gas budget: {actual vs estimated}
- Defects found/fixed: {N}/{M}
- Security findings: {N critical, N high}

### Risks
- {risk: mitigation status}

### Next Sprint
- {planned stories}
```

## Milestone Report Template
```markdown
## Milestone {N}: {Title} — {status}
Duration: {start} → {end}
Deliverables: {N} of {M} complete
Quality: Gate 2 {status}, Gate 3 {status}
Key Achievements: {list}
Remaining Work: {list}
```
