# Security Policy

If you discover a security issue in this repository, report it privately through GitHub Security Advisories or the repository's private vulnerability reporting flow if enabled.

## Scope

This repository contains reusable Copilot and VS Code customization templates. Treat the files as configuration samples and review them before use in your own repository.

## What to Include

- A clear description of the issue
- The affected file or files
- Any reproduction steps
- Any security impact you observed

## Safe Usage Notes

- Do not place secrets in configuration examples or copied templates.
- Review copied scripts and hook commands before installing them in another repository.
- Rotate any credentials that may have been exposed during testing.
